plugin.video.hollandsport
===================

Kodi plugin of Holland Sport

# Installation
1. Download the package as a zip
2. Put the zip on a place which is accessable by the Kodi
3. Go to settings > Addon
4. Install addon using zip
5. Select the zip file
6. Navigate to Videos > addons > HollandSport
7. Green streams are online, red streams are offline
